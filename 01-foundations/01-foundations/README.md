# Stage 1 — Foundations

Building core Python, NumPy, and Pandas skills using real 
UK financial market data.

## Notebooks

### 01 - UK House Prices
- First data analysis notebook
- Bar chart comparing average house prices across UK cities
- **Key concepts:** Pandas DataFrames, Matplotlib bar charts

### 02 - HSBC Stock Analysis (NumPy)
- Real HSBC (HSBA.L) closing price data from LSE
- Daily returns, best/worst days, annualised volatility
- 20-day moving average calculation
- **Key concepts:** NumPy arrays, np.diff, np.convolve, financial maths

### 03 - UK Banking Stocks (Pandas)
- Live data for HSBC, Barclays, Lloyds, NatWest, Standard Chartered
- 6-month performance comparison and ranking
- Annualised volatility and risk analysis
- Correlation matrix — systemic risk in UK banking sector
- **Key concepts:** Pandas DataFrame, pct_change, corr, heatmap
- **Finding:** All 5 banks highly correlated — classic systemic risk
